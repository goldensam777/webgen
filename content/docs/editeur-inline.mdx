title: Édition inline
description: Cliquer sur n'importe quel texte pour le modifier directement.
---

L'édition inline est le moyen le plus rapide de personnaliser le contenu de votre site. Cliquez sur un texte dans la prévisualisation et modifiez-le sur place, sans ouvrir de formulaire.

## Comment ça fonctionne

Dans le canvas de prévisualisation, **tous les textes sont cliquables**. Cliquez sur un titre, une description, un label de bouton ou toute autre zone de texte pour passer en mode édition.

Un **champ de texte** remplace le texte statique. Vous pouvez taper directement votre contenu. Les modifications sont reflétées **instantanément** dans le store du projet — aucune action de sauvegarde manuelle n'est nécessaire.

## Ce qui est éditable inline

Tous les champs textuels des sections sont modifiables :

- Titres et sous-titres (hero, features, pricing, FAQ…)
- Corps de texte et descriptions
- Labels de boutons (CTA, liens)
- Noms, rôles et contenus des témoignages
- Questions et réponses de la FAQ
- Valeurs et labels des stats
- Informations de contact (email, téléphone, adresse)
- Logo textuel dans la navbar et le footer
- Texte de copyright

## Ce qui N'est PAS éditable inline

| Élément | Comment le modifier |
|---|---|
| Couleurs | Panneau **Style** (thème) ou mode **Canvas** |
| Images et avatars | Non supporté dans l'éditeur actuel |
| Liens de navigation | Panneau de sections |
| Structure des sections | Panneau de sections (ajout/suppression/ordre) |

## Raccourcis clavier

| Raccourci | Action |
|---|---|
| `Enter` | Confirmer et quitter l'édition (champs mono-ligne) |
| `Shift+Enter` | Nouvelle ligne (champs multi-lignes) |
| `Esc` | Annuler les modifications et restaurer l'ancien texte |

<Callout variant="tip">
Lorsque vous survolez une section, une **barre d'outils flottante** apparaît en haut. Elle donne accès au mode Canvas, aux options de section, et permet de naviguer rapidement entre les sections sans passer par la barre latérale.
</Callout>

## Sauvegarde automatique

Chaque modification inline est enregistrée en temps réel dans l'état du projet. Si vous publiez ou téléchargez le site après une édition inline, le contenu modifié est bien inclus.
