La qualité du site généré dépend directement de la qualité de votre description. Claude analyse chaque mot pour prendre des décisions de design et de contenu.

## Ce que l'IA fait avec votre description

1. **Détecte le secteur** → choisit une palette de couleurs et une typographie adaptées
2. **Identifie le public cible** → adapte le ton et le vocabulaire du contenu
3. **Sélectionne les sections** → inclut uniquement ce qui est pertinent
4. **Génère du contenu réaliste** → titres percutants, descriptions convaincantes, chiffres plausibles
5. **Ordonne la page** → place les sections dans un ordre logique (hero → features → social proof → pricing → CTA)

## Ce qu'il faut inclure

- **Nom de l'entreprise ou du projet**
- **Secteur d'activité** (SaaS, e-commerce, cabinet médical, restaurant, portfolio…)
- **Public cible** (développeurs, PME, parents, étudiants…)
- **Style souhaité** (minimaliste, coloré, luxe, corporate, fun…)
- **Sections à inclure** (pricing, témoignages, FAQ, formulaire de contact…)
- **Contenu spécifique** (slogan existant, services précis, noms de plans tarifaires…)

## Exemples comparatifs

| Qualité | Exemple |
|---|---|
| Médiocre | "Site pour mon agence" |
| Correct | "Site pour une agence web appelée Pixelcraft" |
| Bon | "Site pour Pixelcraft, agence web créative spécialisée en branding et développement sur mesure pour startups. Style sombre et moderne. Sections : hero, services, portfolio fictif, témoignages, contact." |
| Excellent | "Site pour Pixelcraft, agence créative à Paris fondée en 2019. Nos services : identité visuelle, sites web, applications mobiles. Clients : startups tech et scale-ups. Ton : expert mais accessible. Style : sombre, typographie bold, accents violet. Sections : hero avec slogan fort, 4 services, stats (50+ clients, 120 projets), témoignages, pricing (Starter/Growth/Enterprise), CTA newsletter." |

## Guide de style par secteur

| Secteur | Palette | Police recommandée | Sections typiques |
|---|---|---|---|
| Tech / SaaS | Sombre, bleu/violet | Space Grotesk, Inter | hero, features, pricing, FAQ |
| Restauration | Chaud, beige/terre | Merriweather, Lora | hero, menu fictif, témoignages, contact |
| Santé / Bien-être | Bleu clair, blanc | Nunito, DM Sans | hero, services, stats, témoignages |
| Créatif / Portfolio | Vif, contrasté | Syne, Cabinet Grotesk | hero, portfolio, à propos, contact |
| Finance / Juridique | Sobre, navy/gris | Manrope, Outfit | hero, services, stats, CTA |

<Callout variant="tip">
Soyez **aussi spécifique que possible** sur le contenu : si vous avez de vrais noms de services, de vraies offres tarifaires ou un vrai slogan, incluez-les. Claude utilisera vos données réelles plutôt que d'inventer, ce qui vous évitera de tout réécrire ensuite.
</Callout>

## Longueur optimale

Une description de **3 à 8 phrases** est idéale. En dessous, le résultat sera générique. Au-delà, les informations les moins importantes peuvent être ignorées. Pour les projets complexes, complétez avec des fichiers joints (PDF de brochure, captures d'écran).
