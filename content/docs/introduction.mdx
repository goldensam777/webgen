Webgen est un générateur de sites web propulsé par l'IA. Décrivez votre projet en langage naturel et obtenez en quelques secondes un site complet multi-pages, prêt à publier ou à télécharger.

## Comment ça marche

Le flux de travail tient en trois étapes :

1. **Décrire** — Rédigez une description de votre site (activité, style, sections souhaitées). Plus elle est précise, meilleur est le résultat.
2. **Éditer** — L'IA génère un site complet. Modifiez les textes en cliquant directement dessus, ajustez le thème, réorganisez les sections.
3. **Publier** — Mettez en ligne sur `votre-slug.webgen.app` en un clic, ou téléchargez le site en HTML/ZIP pour l'héberger vous-même.

| Fonctionnalité | Détail |

| Génération IA | Propulsé par Claude — contenu réaliste et mise en page cohérente |

| Export HTML/ZIP | Site 100 % statique, hébergeable partout |
| Thème personnalisable | 7 variables de couleur + 12 polices Google Fonts |
| Multi-pages | Navbar et footer communs, pages indépendantes |

## Ce que l'IA génère

Pour chaque site, Claude sélectionne automatiquement :

- Une **palette de couleurs** adaptée au secteur (tech → sombre, restauration → chaleureux…)
- Une **typographie** cohérente avec le ton souhaité
- Les **sections pertinentes** parmi les 10 disponibles (hero, features, pricing, FAQ, etc.)
- Un **contenu réaliste** (titres, descriptions, témoignages, tarifs fictifs mais plausibles)

<Callout variant="tip">
Pour obtenir le meilleur résultat, décrivez votre site comme si vous l'expliquiez à un designer : nom de l'entreprise, secteur d'activité, public cible, ton souhaité (professionnel, ludique, minimaliste…) et les sections que vous voulez absolument inclure.
</Callout>

## Prérequis

Un compte Webgen suffit. Aucune compétence technique n'est requise — ni HTML, ni CSS, ni JavaScript.
