La création d'un site Webgen commence par une simple description en texte libre. Voici le processus complet.

## Étape 1 — Accéder à la page de création

Depuis le tableau de bord, cliquez sur **Nouveau site** ou naviguez directement vers `/create`. Vous arrivez sur le formulaire de description.

## Étape 2 — Rédiger la description

Le champ de description est le point de départ de tout. Indiquez :

- **Le nom** de votre entreprise ou projet
- **Le secteur d'activité** (SaaS, restauration, cabinet médical, portfolio…)
- **Le public cible** (développeurs, familles, PME…)
- **Le style souhaité** (moderne, minimaliste, coloré, corporate…)
- **Les sections à inclure** (pricing, témoignages, formulaire de contact…)
- **Du contenu spécifique** si vous en avez (slogan, services, offres tarifaires)

### Exemple de bonne description

```
Crée un site pour "NutriCoach", une application mobile de suivi nutritionnel
destinée aux sportifs amateurs. Style moderne et énergique, couleurs vives.
Sections : hero accrocheur, 3 fonctionnalités clés, témoignages d'utilisateurs,
pricing avec 3 plans (Gratuit / Pro / Équipe), et une FAQ.
```

## Étape 3 — Lancer la génération

Cliquez sur **Générer** ou utilisez le raccourci `Cmd+Enter` (Mac) / `Ctrl+Enter` (Windows). La génération prend environ **10 secondes**.

Pendant ce temps, l'IA :
- Analyse le secteur et choisit une palette de couleurs adaptée
- Sélectionne et ordonne les sections pertinentes
- Rédige tout le contenu textuel
- Configure la typographie

## Ce que vous obtenez

À la fin de la génération, vous arrivez directement dans l'éditeur avec :

- Un **site multi-pages** (Accueil + pages supplémentaires selon la description)
- Une **navbar** avec logo et liens de navigation
- Un **hero** personnalisé avec titre, sous-titre et bouton d'appel à l'action
- Les **sections** demandées, dans un ordre logique
- Un **footer** avec informations de contact
- Un **thème cohérent** (couleurs + police)

<Callout variant="note">
L'IA détecte le secteur à partir de votre description et applique automatiquement un style adapté : tech → palette sombre, alimentation → tons chauds, santé → bleu clair, créatif → couleurs vives. Vous pouvez toujours modifier le thème ensuite dans l'onglet **Style** de l'éditeur.
</Callout>

## Régénérer ou affiner

Si le résultat ne vous convient pas, retournez sur `/create` et affinez votre description. Vous pouvez aussi **joindre des fichiers** (PDF, images) pour donner plus de contexte à l'IA.
