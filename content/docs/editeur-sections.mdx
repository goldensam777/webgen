---
title: Gérer les sections
description: Ajouter, supprimer et réorganiser les sections de chaque page.
---

L'éditeur Webgen vous permet de contrôler entièrement la structure de chaque page grâce au panneau de sections dans la barre latérale gauche.

## Le panneau de sections

Dans la barre latérale gauche, l'onglet **Sections** liste toutes les sections présentes sur la page en cours. Chaque section est représentée par une carte avec son nom et des actions disponibles.

## Sections disponibles

Webgen propose 10 types de sections :

| Section | Rôle |
|---|---|
| `navbar` | Barre de navigation principale avec logo, liens et bouton CTA |
| `hero` | Section d'accroche en haut de page (titre, sous-titre, CTA) |
| `features` | Grille de fonctionnalités ou avantages (icônes + texte) |
| `stats` | Chiffres clés en grande typographie |
| `testimonials` | Avis clients avec nom, rôle et contenu |
| `pricing` | Plans tarifaires avec liste de fonctionnalités |
| `faq` | Accordéon de questions/réponses |
| `cta` | Bannière d'appel à l'action centrée |
| `contact` | Informations de contact (email, téléphone, adresse) |
| `footer` | Pied de page avec logo, description et copyright |

## Ajouter une section

1. Dans le panneau de sections, cliquez sur **+ Ajouter une section**
2. Choisissez le type de section dans la liste
3. La section est insérée en bas de la page avec un contenu par défaut

## Supprimer une section

Survolez la carte de la section dans le panneau, puis cliquez sur l'icône **corbeille**. La suppression est immédiate (sans confirmation — utilisez `Ctrl+Z` pour annuler si disponible).

## Réorganiser par glisser-déposer

Dans le panneau de sections, faites glisser une carte par son icône de poignée (⠿) pour changer l'ordre d'affichage. Le canvas se met à jour en temps réel.

<Callout variant="note">
La `navbar` et le `footer` sont des sections globales — elles apparaissent sur toutes les pages du site. Les modifier sur une page les modifie partout. Les autres sections sont propres à chaque page.
</Callout>

## Gestion multi-pages

Webgen supporte plusieurs pages par site. La liste des pages est accessible en haut de la barre latérale. Chaque page possède ses propres sections (hors navbar et footer). Pour ajouter une page, cliquez sur **+ Nouvelle page** dans le sélecteur de pages.
