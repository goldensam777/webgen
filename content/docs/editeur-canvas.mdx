---
title: Mode Canvas
description: Sélectionner et styliser les éléments individuels d'une section.
---

Le mode Canvas permet d'aller plus loin que le thème global en stylisant chaque élément d'une section de façon indépendante — fond, typographie, espacement, visibilité.

## Activer le mode Canvas

Survolez une section dans le canvas de prévisualisation. Une **barre d'outils flottante** apparaît en haut de la section. Cliquez sur l'icône **grille** (⊞) pour activer le mode Canvas de cette section.

La section passe en mode d'édition fine : les éléments individuels deviennent sélectionnables.

## Sélectionner un élément

En mode Canvas, survolez n'importe quel élément (titre, bouton, carte, image, badge…). Un **contour vert** indique les éléments survolés ou sélectionnés. Cliquez pour sélectionner un élément et faire apparaître sa barre d'outils.

## La barre d'outils d'élément

Lors de la sélection d'un élément, une barre d'outils contextuelle apparaît avec les propriétés modifiables :

| Propriété | Description |
|---|---|
| Couleur de fond | Arrière-plan de l'élément (indépendant du thème) |
| Couleur de texte | Couleur du texte contenu dans l'élément |
| Border radius | Arrondi des coins (en px) |
| Padding | Espacement interne (haut/bas/gauche/droite) |
| Font weight | Graisse du texte (400 normal, 600 semi-bold, 700 bold…) |
| Font size | Taille de police (en px ou rem) |
| Opacité | Transparence de l'élément (0–100 %) |
| Masquer sur mobile | Cache l'élément sur les petits écrans |

Chaque modification est appliquée en temps réel et sauvegardée automatiquement.

## Quitter le mode Canvas

Cliquez à nouveau sur l'icône grille dans la barre d'outils flottante, ou cliquez en dehors de la section pour désactiver le mode Canvas.

<Callout variant="note">
Le mode Canvas opère au niveau de l'**élément individuel**, tandis que le panneau **Style** (thème) opère au niveau du site entier. Les styles Canvas sont prioritaires sur le thème global — modifier la couleur primaire du thème ne remplacera pas une couleur définie manuellement en mode Canvas.
</Callout>

## Cas d'usage typiques

- Donner un fond coloré spécifique à une carte de tarification
- Agrandir le titre hero au-delà de la taille par défaut
- Masquer un élément décoratif sur mobile
- Ajuster l'opacité d'un badge pour un effet subtil
