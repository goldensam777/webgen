# Webgenx — IA Website Builder (SaaS Ready)

Webgenx est un générateur de sites web professionnel piloté par l'IA (Claude 3.5 Sonnet / Gemini 2.0 Flash). Il permet de transformer une description textuelle ou des documents (PDF, CSV, MD) en une présence en ligne complète, personnalisable et publiable en moins de 30 secondes.

## 🚀 Fonctionnalités Clés (Refonte 2026)

### 🎨 Design & Expérience (Pro)
- **Interface Immersive** : Design moderne utilisant le glassmorphism, les grilles Bento et un mode sombre adaptatif.
- **Éditeur "Studio"** : Un canevas fluide avec prévisualisation en temps réel (Desktop/Mobile) et édition directe (Inline).
- **Magic Patch (Ctrl+I)** : Retouches intelligentes assistées par l'IA pour modifier la structure ou le style par simple commande vocale/textuelle.

### 💼 Outils Business (SaaS Ready)
- **Génération de Leads** : Formulaires de contact dynamiques avec stockage et gestion des messages dans le Dashboard (mini CRM).
- **Support Client IA** : Chatbot intelligent intégré, configuré automatiquement avec le contenu du site pour répondre aux visiteurs.
- **SEO Avancé** : Panneau de configuration complet pour les titres, descriptions, indexation et OpenGraph (réseaux sociaux).
- **Système de Plans** : Préparation technique pour la monétisation (Plans Free & Pro).

### 🛠️ Technologie
- **Frontend** : Next.js 16 (App Router), React 19, Tailwind CSS 4.
- **Backend & Auth** : Supabase (Database, Auth, Storage).
- **IA Engines** : Anthropic Claude 3.5 Sonnet (Génération), Gemini 2.0 Flash (Analyse de documents & Chat).
- **Export** : Téléchargement du code source complet (ZIP) ou déploiement instantané.

## 📁 Structure du Projet
- `/app` : Pages de l'application (Dashboard, Create, Settings, Preview, s/[slug]).
- `/components` : Bibliothèque de composants UI et sections du site.
- `/lib` : Logique métier (IA, Schémas, Bridge Preview, Auth).
- `/data` : Données de configuration et types.
