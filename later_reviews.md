# Later Reviews

## Option B — Éditeur Canva niveau élément

Quand l'Option A (éditeur niveau section) est stable, implémenter un vrai éditeur visuel :

- Cliquer sur un texte précis dans le preview → édition inline directement
- Cliquer sur une image → la déplacer ou la remplacer
- Handles de resize sur les éléments individuels
- Liberté de positionnement pixel par pixel
- Drag & drop d'éléments à l'intérieur d'une section

**Référence :** Canva, Framer (mode libre), Penpot
